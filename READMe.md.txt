# AI Resume Optimizer

An AI-powered tool that improves resumes by giving feedback and rewriting sections.

## What it does
- User pastes a resume
- AI analyzes it
- Returns improvements like:
  - Better summary
  - Stronger bullet points
  - Skill suggestions
  - Resume score

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- AI: OpenAI API

## How to run it (WIP)
Frontend:
npm install
npm start

Backend:
npm install
node server.js
